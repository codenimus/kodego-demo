Temperature Converter (Using function)

Created a HTML page that converts Fahrenheit to Celsius and back

Convert Temperature from °F to °C

to Celsius

Convert Temperature from °C to °F

to Fahrenheit

Solution

// F° to C°
function tempConvert(valNum) {
    valNum = parseFloat(valNum);
    document.getElementById("outputCelsius").innerHTML = (valNum - 32) / 1.8 + "°C";
    }
                                
// C° to F°
    function tempConvert2(valNum) {
    valNum = parseFloat(valNum);
    document.getElementById("outputFahrenheit").innerHTML = (valNum * 1.8) + 32 + "°F";
    }
